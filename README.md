# Sokoban Oyunu

Bu proje, klasik Sokoban bulmaca oyununun Python ve Pygame ile geliştirilmiş bir versiyonudur.

## 🎮 Oyun Özellikleri

- Çoklu seviye sistemi
- Özel iksir mekanizması (kutuları çekme özelliği)
- Görsel arayüz
- Animasyonlu karakter hareketleri
- Menü sistemi

## 🛠️ Gereksinimler

- Python 3.x
- Pygame
